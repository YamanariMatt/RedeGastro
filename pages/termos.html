Termos de uso

Estes termos são fictícios e existem apenas para compor o protótipo navegável da RedeGastro.

O projeto é um protótipo. Não envie informações sensíveis reais.

1. Natureza do projeto

Protótipo visual e navegável

A RedeGastro, nesta versão, é apenas uma demonstração visual criada com HTML, CSS e JavaScript puro para validar a ideia com potenciais usuários.

O site não representa um sistema operacional real, não possui backend, banco de dados, autenticação real, pagamentos, contratação ou qualquer processamento real de informações.

2. Contratações

Nenhuma contratação real

Nenhuma contratação real é feita pela plataforma nesta versão. Vagas, profissionais, restaurantes, avaliações, interesses e contatos exibidos no protótipo são fictícios e servem apenas para demonstração.

3. Dados demonstrativos

Dados fictícios

Todos os dados exibidos no protótipo são simulados. Nomes, avaliações, vagas, métricas, dashboards e informações administrativas foram criados para representar possíveis telas futuras da RedeGastro.

4. Informações sensíveis

Não envie dados reais

Os formulários existentes são apenas demonstrativos. Não envie CPF, documentos, senhas, dados financeiros, informações pessoais sensíveis ou qualquer conteúdo que você não queira usar em um protótipo.

Campos como CPF, WhatsApp, login e senha existem apenas para testar a experiência visual e validações locais no navegador.

5. Contato externo

WhatsApp simulado

Os botões de WhatsApp usam um número fictício e mensagens prontas para demonstrar como o contato poderia funcionar. Não há integração real, registro de contato ou garantia de comunicação com qualquer pessoa ou empresa.

Resumo

Esta página não substitui termos jurídicos reais. Ela existe apenas para deixar claro, dentro do protótipo, que nada nesta versão tem efeito operacional, contratual ou comercial.

Esta versão não possui

  • Login real
  • Banco de dados
  • Pagamento
  • Contratação real
  • Validação real de documentos
