Política de privacidade

Esta política é fictícia e foi criada para explicar o comportamento do protótipo navegável da RedeGastro.

O protótipo não coleta dados reais e os formulários não salvam informações.

1. Coleta de dados

O protótipo não coleta dados reais

A versão publicada no GitHub Pages é estática e não possui backend, banco de dados ou servidor próprio para armazenamento de informações enviadas pelos usuários.

Qualquer dado digitado nos formulários é usado apenas no navegador para simular validações visuais.

2. Formulários

Formulários não salvam informações

Os formulários de cadastro e login bloqueiam o envio real e exibem mensagens demonstrativas. Eles não criam contas, não autenticam usuários e não armazenam dados em banco.

3. CPF e documentos

CPF é apenas um campo demonstrativo

O campo de CPF existe apenas para demonstrar como uma validação matemática poderia aparecer em uma versão futura. Não envie CPF real, documentos pessoais ou informações sensíveis neste protótipo.

4. Preferências locais

Uso pontual de LocalStorage

O protótipo pode usar LocalStorage apenas para simular favoritos ou escolhas visuais neste navegador. Essas informações não são enviadas para servidores e podem ser apagadas limpando os dados do navegador.

5. Versão futura

LGPD e segurança

Uma versão real da RedeGastro deverá seguir a Lei Geral de Proteção de Dados, definir bases legais de tratamento, políticas de retenção, controles de segurança, consentimentos, direitos dos titulares e canais de atendimento.

Esses elementos ainda não existem nesta versão demonstrativa.

Resumo

Esta versão é apenas uma simulação visual. Nenhum dado é salvo, nenhum login é real e nenhuma contratação acontece por esta versão.

Cuidados

  • Não informe CPF real
  • Não envie documentos
  • Não use senhas reais
  • Não informe dados bancários
  • Use apenas dados fictícios
